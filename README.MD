# Python Studies – Estruturas de Dados

Este repositório contém exercícios práticos focados nos fundamentos de Python, com ênfase em estruturas de dados essenciais para análise e engenharia de dados.

---

## Conteúdo Estudado

### Listas

* Criação e manipulação de listas
* Acesso a elementos
* Ordenação (crescente e decrescente)
* Filtros (ex: números pares)
* Contagem de elementos
* Inversão de listas
* Operações matemáticas com elementos

---

### Tuplas

* Criação de tuplas
* Acesso por índice
* Verificação de elementos
* Conversão entre lista e tupla
* Uso de funções como `max()` e `min()`
* Concatenação
* Imutabilidade (tentativa de alteração)

---

### Dicionários

* Estrutura chave → valor
* Acesso a dados
* Adição, atualização e remoção de elementos
* Percorrer chaves e valores
* Verificação de existência
* Cálculo de média
* Busca de valores com `input()`

---

### Conjuntos (Sets)

* Criação de conjuntos
* União, interseção e diferença
* Adição e remoção de elementos
* Verificação de subconjuntos e disjunção
* Remoção de duplicados
* Conversão de lista para conjunto
* Contagem de elementos únicos

---

## Objetivo

Consolidar a base de Python.
